# Museum


# How do I run it?

You'll need to install the dependencies and build the application

## Install dependencies
```sh
npm i
```

## Building the repo

```sh
// build and bundle the app
npm run build_bundle
// run a dev server
npm run start
```

## Type-checking the repo
```sh
npm run type-check
```

```sh
npm run type-check -- --watch
```
